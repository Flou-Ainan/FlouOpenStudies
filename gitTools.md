## Git tools 

#List branches:
git branch

#List all branches (remotes an locals)
git branch -a

#Create branch:
git branch -C "name"

#Delete branch:
git branch --delete "name"

#Push new branch to Romote
git push -u 'remote-name' 'branch-name' 
- the -u is for upstream

#Add a remote
git remote add "remote-name" "https://remote.domain/repo.git"
